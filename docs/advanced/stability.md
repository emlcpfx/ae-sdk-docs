# Stability

> 1 Q&A · source: AE plugin dev community Discord

### What challenges do GPU plugins face with MFR compatibility?

GPU plugins can experience crashes during long MFR rendering sessions that are difficult to debug. OpenGL-based plugins have particular stability issues, especially on macOS where separate code paths for Metal and Windows may be necessary.
